## 🚀 Quick Start

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/denisjanuzi.git

# Navigate to the project directory
cd denisjanuzi

# Install dependencies
npm install
```

### Development

```bash
# Start development server
npm run dev

# The site will be available at http://localhost:5173
```

### Build

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```

## 🎨 Customization

### Update Personal Information

1. **About Section**: Edit `src/components/AboutSection.vue`
2. **Projects**: Modify the `projects` array in `src/components/ProjectsSection.vue`
3. **Contact**: Update contact methods in `src/components/ContactSection.vue`
4. **Navigation**: Adjust navigation items in `src/components/NavHeader.vue`

### Color Scheme

The color scheme is defined in `tailwind.config.js`:

```javascript
colors: {
  'dark-bg': '#0F172A',        // Main background
  'dark-secondary': '#1E293B', // Secondary background
  'accent-blue': '#3B82F6',    // Primary accent
  'accent-light': '#60A5FA',   // Light accent
}
```
